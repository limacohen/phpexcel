==============================
PHPExcel Spark for CodeIgniter
==============================
This spark is just a package for [**PHPExcel**](https://github.com/PHPOffice/PHPExcel), a library for reading and writing spreadsheet files.

Installation
============
Once loaded, just use $this->excel
